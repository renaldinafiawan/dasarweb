⚡ File: script.js
function showMessage() {
    alert('Halo! Website ini dibuat oleh kelompok kami.');
}

// Smooth scrolling untuk navigasi
document.querySelectorAll('nav a').forEach(anchor => {
    anchor.addEventListener('click', function (e) {
        e.preventDefault();
        const target = document.querySelector(this.getAttribute('href'));
        if (target) {
            target.scrollIntoView({
                behavior: 'smooth'
            });
        }
    });
});

// Menampilkan pesan selamat datang

    console.log('Website berhasil dimuat!');
});

Setup GitHub Repository

🔧 Langkah-langkah:

  1. 1
    Login ke GitHub

    Masuk ke akun GitHub yang sudah dibuat

  2. 2
    Buat Repository Baru

    Klik tombol "New" atau "+" → "New repository"

  3. 3
    Isi Detail Repository
    • • Repository name: my-first-website
    • • Description: "Website pertama kelompok [nama kelompok]"
    • • ✅ Public
    • • ✅ Add a README file
  4. 4
    Klik "Create repository"
📋 Informasi Repository:
4

Upload File ke GitHub

💡 Dua Cara Upload:

Pilih salah satu cara yang paling mudah untuk kelompok kalian

🌐 Cara 1: Melalui Website GitHub

  1. 1. Buka repository yang sudah dibuat
  2. 2. Klik "uploading an existing file"
  3. 3. Drag & drop semua file website
  4. 4. Tulis commit message: "Upload website pertama"
  5. 5. Klik "Commit changes"

⌨️ Cara 2: Menggunakan Git Commands

git clone [URL-repository]
cd my-first-website
# Copy file website ke folder ini
git add .
git commit -m "Upload website pertama"
git push origin main
✅ Checklist Upload:
5

Deploy ke Vercel

🚀 Langkah Deployment:

  1. 1
    Login ke Vercel

    Masuk menggunakan akun GitHub (recommended)

  2. 2
    Import Project

    Klik "New Project" → "Import Git Repository"

  3. 3
    Pilih Repository

    Cari dan pilih repository "my-first-website"

  4. 4
    Configure Project
    • • Project Name: biarkan default atau ganti sesuai keinginan
    • • Framework Preset: pilih "Other" atau biarkan kosong
    • • Root Directory: biarkan default (.)
  5. 5
    Deploy!

    Klik "Deploy" dan tunggu proses selesai

🌍 Setelah Deploy Berhasil:
  • • Kalian akan mendapat URL website (contoh: https://my-first-website-abc123.vercel.app)
  • • Website bisa diakses oleh siapa saja di internet
  • • Setiap kali ada perubahan di GitHub, website otomatis terupdate
